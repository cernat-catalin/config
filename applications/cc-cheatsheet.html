🔄 Workflows & Tips
Plan Mode
  • Cycle: Shift+Tab → Normal → Auto-Accept → Plan
  • Start directly: --permission-mode plan
  • Inline: /plan [description]
Thinking & Effort
  • Toggle: Alt+T
  • Max single-turn effort: type ultrathink
  • View thinking: Ctrl+O (verbose mode)
  • Set level: /effort low / medium / high / max
Git Worktrees new
  • Isolated branch per feature: --worktree name
  • Agent in worktree: isolation: worktree
  • Sparse checkout: sparsePaths setting
  • Auto-create with /batch
Voice Mode
  • Activate: /voice
  • Hold Space to record, release to send
  • Supports 20 languages: EN, ES, FR, DE, CZ, PL…
Context Management
  • Visualize: /context
  • Compress: /compact [focus]
  • Auto-compaction at ~95% capacity
  • Max 1M tokens on Opus 4.6 (Max/Team/Enterprise)
  • CLAUDE.md persists through compaction
Session Power Moves
  • Continue last: claude -c
  • Resume named: claude -r "name"
  • Side question: /btw question
  • Pipe input: cat file | claude -p
SDK / Headless
  • Non-interactive: claude -p "query"
  • JSON output: --output-format json
  • Cost limit: --max-budget-usd 5
  • Bare mode (no hooks/LSP): --bare new
Scheduling & Remote
  • Recurring task: /loop 5m message
  • Remote control: /rc
  • Web session: --remote
  • Cloud schedule: /schedule
